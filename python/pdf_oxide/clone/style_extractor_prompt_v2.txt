## TASK

## PRESET SELECTION RULES

### Table Presets — Decision Rules

If the table has:
- Control ID column + Description + Status columns → "control_matrix"
- Requirement ID + Text + Evidence columns → "requirements_matrix"
- Comparison columns (Feature/Option A/Option B) → "comparison_matrix"
- Date + Amount columns, financial data → "ledger" or "budget_table"
- Term + Definition two-column layout → "glossary"
- Acronym + Expansion two-column layout → "acronyms"
- Bordered grid with header row → "data_grid" (default)
- No borders, whitespace separation → "gridless_report"
- RACI letters (R/A/C/I) in cells → "raci_matrix"
- Risk severity + Likelihood + Impact → "risk_register"
- Test case + Result + Pass/Fail → "test_results"

Valid table_preset values:
["data_grid", "requirements_matrix", "comparison_matrix", "ledger", "sectioned_grid", "gridless_report", "control_matrix", "risk_register", "traceability_matrix", "gap_analysis", "poam", "assessment_results", "audit_findings", "evidence_summary", "compliance_status", "test_results", "parameter_table", "specification_table", "interface_table", "hazard_analysis", "action_items", "decision_log", "change_requests", "schedule_table", "budget_table", "raci_matrix", "meeting_minutes", "revision_history", "glossary", "acronyms", "references", "personnel_table", "distribution_list", "signature_block", "approval_block", "checklist"]

### Header Presets — Decision Rules

If the running header has:
- "NIST" or "Special Publication" text → "nist_header"
- Document title on left, page number on right → "doc_title_header"
- "CONTROLLED" or classification markings → "classified_header"
- Version number visible (v1.0, Rev A) → "versioned_header"
- Chapter name centered → "chapter_header"
- Only page number, no other content → "minimal_header"

Valid header_preset values:
["doc_title_header", "section_header", "classified_header", "versioned_header", "chapter_header", "minimal_header", "dual_logo_header", "nist_header"]

### Footer Presets — Decision Rules

If the running footer has:
- "Page X of Y" format → "page_number_footer"
- Classification marking centered → "classified_footer"
- Document number + Rev + Page → "revision_footer"
- Copyright notice → "copyright_footer"
- Document control number → "doc_control_footer"
- "DRAFT" watermark or text → "draft_footer"
- Only page number, nothing else → "minimal_footer"

Valid footer_preset values:
["page_number_footer", "classified_footer", "revision_footer", "copyright_footer", "doc_control_footer", "draft_footer", "distribution_footer", "minimal_footer"]

### Callout Presets — Decision Rules

If the callout box has:
- Blue background, informational content → "note_box"
- Yellow/orange background, caution text → "warning_box"
- Red background, critical warning → "danger_box"
- Green background, helpful tip → "tip_box"
- Gray background, code/monospace → "code_block"
- Purple background, definition → "definition_box"
- Left border only, quoted text → "quote_block"

Valid callout_preset values:
["note_box", "warning_box", "danger_box", "tip_box", "example_box", "definition_box", "quote_block", "code_block", "requirement_box", "finding_box", "recommendation_box", "reference_box"]

## FONT EXTRACTION RULES

For fonts.body and fonts.heading, use ONLY these ReportLab font names:
- "Helvetica" (sans-serif, similar to Arial)
- "Helvetica-Bold"
- "Times-Roman" (serif, similar to Times New Roman)
- "Times-Bold"
- "Courier" (monospace)
- "Courier-Bold"

If the PDF uses Arial → output "Helvetica"
If the PDF uses Times New Roman → output "Times-Roman"
If the PDF uses Calibri → output "Helvetica"
If uncertain → output "Helvetica"

## COLOR EXTRACTION RULES

Extract hex color codes from the PDF. Look for:
- Table header background color → colors.table_header_bg
- Alternating row color → colors.table_alt_row
- Link/hyperlink color → colors.link
- Heading text color → colors.heading

If a color is not visible or cannot be determined, use the defaults shown in the schema.

## REJECTION CRITERIA — Do NOT include:

- Preset names not in the valid lists above
- Font names not in the ReportLab list above
- Color values that are not valid hex codes (must match #RRGGBB)
- spacing/margin values that are negative
- characteristics.estimated_complexity values other than "simple", "moderate", "complex"

def validate_style_profile(data: dict) -> dict:
    """Validate and fix preset names."""
    VALID_TABLE = set(AVAILABLE_TABLE_PRESETS)
    VALID_HEADER = set(AVAILABLE_HEADER_PRESETS)
    VALID_FOOTER = set(AVAILABLE_FOOTER_PRESETS)
    VALID_CALLOUT = set(AVAILABLE_CALLOUT_PRESETS)

    if data.get("table_preset") not in VALID_TABLE:
        data["table_preset"] = "data_grid"
    if data.get("header_preset") not in VALID_HEADER:
        data["header_preset"] = "doc_title_header"
    if data.get("footer_preset") not in VALID_FOOTER:
        data["footer_preset"] = "page_number_footer"
    if data.get("callout_preset") not in VALID_CALLOUT:
        data["callout_preset"] = "note_box"

    return StyleProfile.from_dict(data)
